## 🇨🇳 简体中文

Item Info Copier 是一个轻量级 Fabric 模组，支持 Minecraft 1.20.1 - 1.20.4。帮助你一键复制物品的详细信息。

**注意**：尽管已标记为与 Minecraft 1.20.1 - 1.20.4 兼容，但此版本仅在 **1.20.1** 上进行过全面测试。

### ✨ 功能特点
- **热键与聊天集成**: 手持物品或在物品栏中悬停时，按下配置的热键（默认为 `Ctrl+C`），会在聊天框中显示一条可交互的详细信息报告。
- **点击复制**: 聊天报告中的每一条信息旁边都有一个 `[复制]` 按钮，方便一键复制。
- **一键复制全部**: 报告中包含一个醒目的 `[点击复制全部信息]` 按钮，用于一次性复制所有可见信息的摘要。
- **声音反馈**: 使用热键或点击聊天按钮复制信息时，会播放音效以供确认。
- **高度可配置**: 通过游戏内配置屏幕（需要 ModMenu）根据你的喜好自定义模组。
- **自定义热键**: 使用全新的、易于使用的按键录制界面，设置你自己的复制热键。提供即时的视觉和声音反馈。
- **信息开关**: 单独启用或禁用以下信息的复制：
  - 物品名称
  - 命名空间 ID (例如, `minecraft:stone`)
  - 翻译键 (例如, `block.minecraft.stone`)
  - 数字 ID
  - NBT 数据
- **控制复制位置**: 分别为手持物品和物品栏中的物品启用或禁用热键。

### 🛠 安装说明
1. 安装 [Fabric Loader](https://fabricmc.net/)。
2. 将 `.jar` 文件放入 `mods` 文件夹。
3. 需要安装 [Cloth Config](https://modrinth.com/mod/cloth-config) 和 [ModMenu](https://modrinth.com/mod/modmenu)。
