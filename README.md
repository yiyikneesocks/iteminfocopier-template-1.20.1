Item Info Copier is a lightweight Fabric mod designed for Minecraft 1.20.1 - 1.20.4. It allows developers and players to quickly copy item details to the clipboard.

### âœ¨ Features
- **Smart Copy**: Press `Ctrl + C` while hovering over an item or holding it.
- **Detailed Output**: Provides ID, NBT, Translation Key, etc.
- **Configuration**: Toggle features via ModMenu.

### ðŸ›  Installation
1. Install [Fabric Loader](https://fabricmc.net/).
2. Place the `.jar` into your `mods` folder.
3. Requires [Cloth Config](https://modrinth.com/mod/cloth-config) and [ModMenu](https://modrinth.com/mod/modmenu).
